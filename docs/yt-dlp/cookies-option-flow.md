# `--cookies` 选项代码调用流程

本文档描述了 yt-dlp 中 `--cookies` 命令行选项的代码实现和调用流程。

---

## 1. 选项定义

**文件**: `yt_dlp/options.py` (第 1538 行)

```python
filesystem.add_option(
    '--cookies',
    dest='cookiefile', metavar='FILE',
    help='Netscape formatted file to read cookies from and dump cookie jar in')
```

**相关选项**:
- `--no-cookies`: 禁用 cookies 文件读写
- `--cookies-from-browser`: 从浏览器提取 cookies

---

## 2. 调用流程图

```
用户传入 --cookies cookies.txt
        ↓
options.py 解析 → params['cookiefile'] = 'cookies.txt'
        ↓
YoutubeDL.__init__(params) 初始化
        ↓
YoutubeDL.cookiejar 属性访问 (cached_property)
        ↓
调用 load_cookies(cookiefile, cookiesfrombrowser, ydl)
       (yt_dlp/cookies.py 第 93 行)
        ↓
创建 YoutubeDLCookieJar(cookie_file) 实例
       (cookies.py 第 107 行)
        ↓
jar.load() 加载 cookies
       (cookies.py 第 108 行)
        ↓
返回合并后的 cookiejar 实例
        ↓
YoutubeDL.cookiejar 被缓存并用于后续 HTTP 请求
```

## 3. 核心组件

### 3.1 CookieJar 属性

**文件**: `yt_dlp/YoutubeDL.py` (第 4179 行)

```python
@functools.cached_property
def cookiejar(self):
    """Global cookiejar instance"""
    try:
        return load_cookies(
            self.params.get('cookiefile'), 
            self.params.get('cookiesfrombrowser'), 
            self)
    except CookieLoadError as error:
        cause = error.__context__
        self.report_error(str(cause), tb=''.join(traceback.format_exception(cause)))
        raise
```

### 3.2 load_cookies 函数

**文件**: `yt_dlp/cookies.py` (第 93 行)

```python
def load_cookies(cookie_file, browser_specification, ydl):
    try:
        cookie_jars = []
        if browser_specification is not None:
            browser_name, profile, keyring, container = _parse_browser_specification(*browser_specification)
            cookie_jars.append(
                extract_cookies_from_browser(browser_name, profile, YDLLogger(ydl), keyring=keyring, container=container))

        if cookie_file is not None:
            is_filename = is_path_like(cookie_file)
            if is_filename:
                cookie_file = expand_path(cookie_file)

            jar = YoutubeDLCookieJar(cookie_file)
            if not is_filename or os.access(cookie_file, os.R_OK):
                jar.load()
            cookie_jars.append(jar)

        return _merge_cookie_jars(cookie_jars)
    except Exception:
        raise CookieLoadError('failed to load cookies')
```

## 6. Cookie 文件格式

yt-dlp 使用 Netscape cookie 格式，示例：

```
# Netscape HTTP Cookie File
# This file is generated by yt-dlp.  Do not edit.

.example.com	TRUE	/	FALSE	1234567890	cookie_name	cookie_value
```

字段说明（制表符分隔）：
1. `domain_name`: 域名
2. `include_subdomains`: TRUE/FALSE
3. `path`: 路径
4. `https_only`: TRUE/FALSE
5. `expires_at`: 过期时间戳
6. `name`: cookie 名称
7. `value`: cookie 值
